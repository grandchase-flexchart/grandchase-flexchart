Grand Chase Flex Chart Help

Click icons to cycle states:

→ Not Acquired
→ Acquired
→ Descent (Up to Level 10)

Note: Clicking more when the character is in Descent will increase its Descent Level. Level 10 will show X. Clicking the icon when at Level 10 will reset the icon to "Not Acquired"

Note: Characters without Descent levels will revert back to "Not Acquired" state when clicked in "Acquired" state



💾 Save - gives you a save code
📂 Load - use a save code to continue progress
🧹 Clear - resets everything
📸 Capture - exports image of the page
